Para abrir a aplicação deve abrir "index.html", vai ser enviado para a home da app, e a partir daí vai conseguir navegar pela a aplicação e disfrutar de todas as funcionalidades disponíveis.

A nossa aplicação está dividida por:

⦁Index - Aqui é a página principal onde se pode navegar para as coleções;

⦁colecoes - Aqui tem as coleções disponíveis, o objetivo é poder organizar melhor as plantas que possui por divisões, espécies, datas etc..;

⦁remove_colecao - Aqui remove se as coleções que pretende eliminar;

⦁add_colecao - Nesta parte adiciona se as coleções que quer adicionar;

⦁colecao - Nesta página é onde vai estar as plantas de cada coleção escolhida anteriormente;

⦁add_planta - Aqui adiciona-se as novas plantas que se quer adicionar;

⦁remove_planta - Aqui vai ser possível remover as plantas de uma coleção;

⦁planta - Aqui vai se conseguir os dados da planta como foto, data, espécie, níveis de agua, fertilização e luz solar, pode se também programar a rega escolhendo manual ou automática;

⦁programar_rega - Aqui vai se poder escolher entre rega manual e automática escolhendo as horas;

⦁diario_verde - Nesta parte o utilizador vai conseguir visualizar os eventos de uma planta como primeira flor, inicio, troca de vaso etc.., distribuídos por um calendário e por "cards" que dá para passar dinamicamente os cartões;

⦁add_evento - Aqui pode se adicionar um evento de uma planta; 

⦁remove_evento - Aqui consegue-se eliminar um evento visualizando os eventos no calendário;

⦁evento - Aqui consegue se ver as informações de um evento especifico e tem a possibilidade de partilhar este evento numa rede social.



Nesta aplicação:
⦁As definições não foram desenvolvidas mostrando sempre um popup quando se carrega no ícone das definições a informar;

⦁Sempre que não é adicionado uma foto nas coleções, plantas ou eventos é sempre adicionado uma foto padrão;

⦁Não dá para ir para a pagina de remover se não existir uma coleção, foto ou evento para ser removida;
